Table of contents

    Directory Structure

    Directory Contains
    application All files needed to build an application
    application\controllers The application controllers (see: Controllers)
    application\models The application models (see: Models)
    application\views The application view files (see: Views)
    assets Additions public available files
    assets\css Stylesheets
    assets\i18n Language (translation) files
    assets\img Images
    config Configuration files for the application and database, and the routes file
    core This is the core of the framework
    core\helpers All autoloaded helper files
    core\modules All autoloaded modules
    templates Templates which are used by the base view class
