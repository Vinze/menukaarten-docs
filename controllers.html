Controllers

Controllers are loaded by the router and default to the following format: http://www.website.com/<controller>/<action>/<id> For example, to view a user with the id 32, the following url should work: http://www.website.com/users/view/32 This will call the controller Users_Controller, the action action_view and will set the id variable in the router. To get the requested id, all you have to do is: $user_id = get(‘id’) Redirect When redirecting to another page, call the following action from the controller: $this->redirect(‘users/index’) Or to redirect to an external website: $this->redirect(‘http:://www.google.com/’)
