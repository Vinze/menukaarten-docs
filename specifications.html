    Technical specifications

    The system was tested an works without any issues on a server with these specifications.

    Webserver: Apache
    PHP version: 5.4 and higher
    PHP extensions required mod_rewrite (needed for URL rewriting)
    gd (needed for image editing/resizing)
    Database: MySQL 5.4 and higher
    MySQL extensions required MySQLi
