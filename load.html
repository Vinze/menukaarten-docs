    Load

    To load certain files and classes we've build an special load class. The load class includes the needed file(s), initializes the instance(s) when needed, and return the object.

    Loading views

    The view classes is automatically loaded in the base_controller. In case you want to you an other template you have to create a new instance of the view class. This can be done with:

    // Load a view object with the default template
$view = load::view();

// Load a view object with no template
$view = load::view(null)

// Load a view object with the dashboard.php template
$view = load::view('dashboard')

    Loading models

    When you need a new instance of a model, you can call load::model.

    $user = load::model('User'); // Returns a new User model

    Initialize the database connection

    The base model uses a MySQLi object for communication with the database. Because this framework uses the singleton design pattern there is a special function for loading the MySQLi object.

    $mysqli = load::database();
