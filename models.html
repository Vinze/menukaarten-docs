Models

Finding rows

->find() To find a user with a certain id: $user->find(232) $user->find(2) Find all users from the database $user->find(‘all’) Or: $user->find() Find the first user from the database $user->find(‘one’)

Finding where

->where() Find one user who’s firstname is Jan: $user->where(firstname = ?’, ‘Jan’)->find(‘one’) Find all users who’s firstname can be multiple values: $user->where(‘firstname = ?’, array(‘Jan’, ‘Piet’, ‘Dirk’)->find(‘all’) Find users with a certain first and lastname: $user->where('firstname = ? AND lastname = ?', array('Jan', 'de Boer'))->find(‘all’);

Selecting fields

->select() By default, the model return all fields with *. Optionally you can specify with field you want to return bij using the select function. You can give it an array with the fields you want, example: $user->select(‘firstname’, age’, ‘hometown’)->find(‘all’)

Sorting the results

->sort() To sort all users by lastname and then firstname, use: $user->sort(‘lastname ASC, firstname ASC)->find() To sort the users by age, oldest first: $user->sort(‘age DESC’)->find(‘all’)

Limit the results

->limit() To limit the number of results, you can specify a limit. Get the users 10 to 20: $user->limit(10, 10)->find(‘all’) Get the users 0 to 50: $user->limit(0, 50)->find(‘all’) Get the users 50 to 75: $user->limit(50, 25)->find(‘all’)
