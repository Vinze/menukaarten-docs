    HTML helper

    Anchor

    To create a anchor (link), you can use the following function. This function will automatically add the attribute target="_blank" when the link contains http:// of https://. So links on your own website can me relative from the base path.

    html::a($link, $name, $attributes);
    Parameters:
    • $link string required
    • $name string optional
    • $attributes array optional
    Example: 
    echo html::a('users/new', 'New user'));
// Returns:
<a href="http://example.com/users/new">New user</a>

echo html::a('http://www.google.com', 'Google'));
// Returns:
<a href="http://www.google.com" target="_blank">Google</a>

    Image

    To include an image use this function. It will automatically find out it's dimensions and add it as attributes. When no $name has been given the alt attribute will contain the image URL.

    html::image($link, $name, $attributes);
    Parameters:
    • $link string required
    • $attributes array optional
    Example: 
    echo html::image('assets/img/car.jpeg', 'My car');
// Returns:
<img src="http://example.com/assets/img/car.jpeg" height="504" width="290" name="My car" alt="My car">

    Stylesheet

    Create a link to the stylesheet.

    html::stylesheet($link, $attributes);
    Parameters:
    • $link string required
    • $attributes array optional
    Example: 
    echo html::stylesheet('assets/css/dashboard.css'));
// Returns:
<link href="http://example.com/assets/css/stylesheet.css" rel="stylesheet" type="text/css" />

    Javascript

    Create a link to the Javascript file.

    html::javascript($link, $attributes);
    Parameters:
    • $link string required
    • $attributes array optional
    Example: 
    echo html::javascript('assets/js/jquery.js'));
// Returns:
<script type="text/javascript" src="http://example.com/assets/js/jquery.js"><script>
